# SummaryTranscription

## 概要
SummaryTranscription は、授業や講義の音声を  
**リアルタイムで文字起こし**し、  
**重要なポイントを簡易的に要約して表示する**  
教育向け・情報保障支援プロトタイプです。

API課金を使わず、Mac上のローカル環境のみで動作します。

---

## 背景・目的
聴覚障がいのある学生や、  
話を聞きながら情報を整理することが苦手な学生に対して、  
授業内容を「文字」と「要点」で同時に提示することで、  
学習理解を支援することを目的としています。

本リポジトリは、  
**技術的な実現可能性を確認するためのプロトタイプ**です。

---

## 主な機能
- マイク入力による音声取得
- Whisper を用いた日本語のリアルタイム文字起こし
- 一定時間ごとに重要語を抽出する簡易要約
- ターミナル上で字幕と要約を同時表示

---

## 動作環境
- macOS
- Python 3.9.x
- マイク付きPC

---

## セットアップ方法

### 1. 必要なライブラリのインストール
```bash
pip3 install -r backend/requirements.txt

2. 音声処理ツールのインストール
brew install ffmpeg

実行方法
python3 backend/realtime_transcribe_with_summary.py


実行後、マイクに向かって話すと字幕が表示されます

Ctrl + C で終了します
